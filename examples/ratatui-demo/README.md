# ratatui-demo

A lived-in example of what your crate looks like after running `ono add`.

The entire `src/ono/` subtree was generated by:

```
ono add splash
ono add boot
ono add dashboard
ono add statusbar
```

Nothing in `src/main.rs` imports from the `ono` crate — every component
lives in your own tree and is yours to modify.

## Run it

```
cargo run -p ratatui-demo
```

Splash plays for 3 s, boot for ~7 s, then a live dashboard runs until you
press `q`, `Esc`, or `Ctrl+C`.

## Regenerating

Delete `src/ono/` and re-run the four `ono add` commands above. Pass
`--theme retro|minimal|cyber` on the first add to pick a different
palette for the generated `theme.rs` (it's reused on subsequent adds).
